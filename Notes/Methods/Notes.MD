**METHODS IN JAVA**

-Syntax:
    '

    accessModifier non-accessModifier returnTYpe methodName(type1 var1, type2 var2)

    public static int sum(int a, int b){}

-Method definition: 
    '

    modifier returnType nameOfMethod(paramList){
        //method body
        //defines what method does with its statements
        //basically the code running whenever our method is called.
    }

-Method names usually use camelCase.
-If the return type is not void, return must be used.
